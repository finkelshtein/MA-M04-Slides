Random Variables

Probability & Statistics for Data Science — Lecture 02

0.1 Random Variables

0.2 What is a Random Variable?

A random variable (RV) \(X\) is a function: \[X : \Omega \longrightarrow \mathbb{R}\]

It maps each outcome to a real number.

Roll two dice. Let $X = $ sum of faces.
\(X\) takes values \(2, 3, \ldots, 12\) — each with a different probability.

RVs allow us to do algebra on probabilistic quantities.

0.3 Discrete vs Continuous

Discrete Continuous
Values Countable set Interval in \(\mathbb{R}\)
Summary PMF \(p(x)\) PDF \(f(x)\)
Normalisation \(\sum_x p(x) = 1\) \(\int f(x)\,dx = 1\)
Examples Counts, categories Heights, times

0.4 Probability Mass Function (PMF)

For a discrete RV: \(p(x) = P(X = x)\)

Requirements:

  • \(p(x) \ge 0\) for all \(x\)
  • \(\sum_{x} p(x) = 1\)

Fair die: \(p(x) = \tfrac{1}{6}\) for \(x \in \{1,\ldots,6\}\), zero elsewhere.

0.5 Probability Density Function (PDF)

For a continuous RV, \(P(X = x) = 0\) for any single \(x\).

Instead we use the PDF \(f(x)\): \[P(a \le X \le b) = \int_a^b f(x)\,dx\]

Requirements:

  • \(f(x) \ge 0\)
  • \(\int_{-\infty}^{\infty} f(x)\,dx = 1\)

0.6 Cumulative Distribution Function

The CDF works for both discrete and continuous RVs:

\[F(x) = P(X \le x)\]

Key properties:

  • Non-decreasing: \(x_1 < x_2 \Rightarrow F(x_1) \le F(x_2)\)
  • Right-continuous
  • \(\lim_{x \to -\infty} F(x) = 0\), \(\lim_{x \to +\infty} F(x) = 1\)

For continuous RVs: \(f(x) = F'(x)\)

0.7 Expectation

0.8 Expectation (Mean)

The expected value (mean) of \(X\):

\[\mu = E[X] = \begin{cases} \displaystyle\sum_x x\,p(x) & \text{discrete} \\[8pt] \displaystyle\int_{-\infty}^{\infty} x\,f(x)\,dx & \text{continuous} \end{cases}\]

\(E[X]\) is the long-run average — it may not be an attainable value of \(X\).

0.9 Linearity of Expectation

For constants \(a, b\) and RVs \(X, Y\):

\[E[aX + b] = a\,E[X] + b\]

\[E[X + Y] = E[X] + E[Y] \quad (\text{always!})\]

Note

The second equality holds regardless of dependence between \(X\) and \(Y\).
This makes expectation extraordinarily useful.

0.10 Variance & Standard Deviation

Variance measures spread around the mean:

\[\operatorname{Var}(X) = E\!\left[(X - \mu)^2\right] = E[X^2] - \mu^2\]

Standard deviation: \(\sigma = \sqrt{\operatorname{Var}(X)}\) (same units as \(X\))

Useful identity: \[\operatorname{Var}(aX + b) = a^2\,\operatorname{Var}(X)\]

0.11 Higher Moments

The \(k\)-th moment of \(X\): \(E[X^k]\)

The \(k\)-th central moment: \(E[(X - \mu)^k]\)

Moment Symbol Measures
1st central \(0\) (always zero)
2nd central \(\sigma^2\) Spread
3rd standardised Skewness \(\gamma_1\) Asymmetry
4th standardised Kurtosis \(\gamma_2\) Tail heaviness

Data scientists use skewness and kurtosis to detect non-Normality.

0.12 Moment Generating Functions

The MGF of \(X\): \[M_X(t) = E[e^{tX}]\]

Why useful?

  • \(E[X^k] = M_X^{(k)}(0)\) — moments by differentiation
  • If two RVs share an MGF, they share a distribution
  • Sums of independent RVs: \(M_{X+Y}(t) = M_X(t)\,M_Y(t)\)

0.13 Joint Distributions

0.14 Joint & Marginal Distributions

For two RVs \((X, Y)\):

Joint PMF/PDF: \(p(x, y)\) or \(f(x, y)\)

Marginal distributions by summing/integrating out: \[p_X(x) = \sum_y p(x,y), \qquad f_X(x) = \int f(x,y)\,dy\]

Conditional distribution: \[f_{Y \mid X}(y \mid x) = \frac{f(x,y)}{f_X(x)}\]

0.15 Covariance & Correlation

\[\operatorname{Cov}(X, Y) = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - \mu_X\mu_Y\]

Pearson correlation (dimensionless, \(\in [-1, 1]\)): \[\rho_{XY} = \frac{\operatorname{Cov}(X,Y)}{\sigma_X\,\sigma_Y}\]

If \(\rho_{XY} = 0\), are \(X\) and \(Y\) independent? Not necessarily!
Independence \(\Rightarrow\) zero correlation, but not vice versa.

0.16 Key Takeaways

  • A RV maps outcomes to real numbers; described by PMF/PDF and CDF
  • Expectation is a linear operator — powerfully simple
  • Variance measures spread; standard deviation restores units
  • Covariance captures linear association; correlation standardises it
