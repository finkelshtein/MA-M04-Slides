0.2 Random Variables

Formalising uncertainty: moving from sample spaces to real-valued quantities that enable calculation.

0.3 Random Variables — Definition

Definition — Random Variable A random variable \(X\) is a measurable function \(X : \Omega \to \mathbb{R}\), mapping outcomes \(\omega \in \Omega\) from a probability space \((\Omega, \mathcal{F}, \mathbb{P})\) to real numbers. For any Borel set \(B \subseteq \mathbb{R}\): \[\mathbb{P}(X \in B) := \mathbb{P}(\{\omega \in \Omega : X(\omega) \in B\}).\]

0.3.1 Discrete RV

Takes values in a countable set \(\{x_1, x_2, \ldots\}\). Described by a probability mass function (PMF):

\[p_X(x) = \mathbb{P}(X = x), \qquad \sum_{x} p_X(x) = 1.\]

0.3.2 Continuous RV

Described by a probability density function (PDF) \(f_X\) such that:

\[\mathbb{P}(a \le X \le b) = \int_a^b f_X(x)\,dx, \qquad \int_{-\infty}^{\infty} f_X(x)\,dx = 1.\]

Note The cumulative distribution function (CDF) \(F_X(x) = \mathbb{P}(X \le x)\) exists for both types and is right-continuous, non-decreasing, with \(F_X(-\infty)=0\) and \(F_X(+\infty)=1\).

0.4 Expectation, Variance & Moments

Expectation \[\mathbb{E}[X] = \begin{cases} \sum_x x\, p_X(x) & \text{discrete}\\[4pt] \int_{-\infty}^{\infty} x\, f_X(x)\,dx & \text{continuous} \end{cases}\]

For \(g(X)\): \(\mathbb{E}[g(X)] = \int g(x) f_X(x)\,dx\).

Linearity of Expectation For any constants \(a, b\) and RVs \(X, Y\) (not necessarily independent): \[\mathbb{E}[aX + bY] = a\,\mathbb{E}[X] + b\,\mathbb{E}[Y].\]

Variance & Standard Deviation \[\operatorname{Var}(X) = \mathbb{E}[(X - \mu)^2] = \mathbb{E}[X^2] - \mu^2\] where \(\mu = \mathbb{E}[X]\). \(\operatorname{SD}(X) = \sqrt{\operatorname{Var}(X)}\).

Variance Algebra - \(\operatorname{Var}(aX+b) = a^2\operatorname{Var}(X)\) - \(\operatorname{Var}(X+Y) = \operatorname{Var}(X)+\operatorname{Var}(Y)+2\operatorname{Cov}(X,Y)\) - If \(X \perp Y\): \(\operatorname{Var}(X+Y) = \operatorname{Var}(X)+\operatorname{Var}(Y)\)

Worked Example Let \(X \sim \text{Bernoulli}(p)\). Then \(\mathbb{E}[X] = p\) and \(\operatorname{Var}(X) = p(1-p)\), maximised at \(p = \tfrac{1}{2}\). For \(Y = \sum_{i=1}^n X_i\) i.i.d.: \(\mathbb{E}[Y] = np\), \(\operatorname{Var}(Y) = np(1-p)\).

0.5 Key Distributions in Data Science

Distribution PMF / PDF Mean Variance Use Case
Bernoulli\((p)\) \(p^x(1-p)^{1-x}\) \(p\) \(p(1-p)\) Binary outcomes
Binomial\((n,p)\) \(\binom{n}{x}p^x(1-p)^{n-x}\) \(np\) \(np(1-p)\) Count of successes
Poisson\((\lambda)\) \(e^{-\lambda}\lambda^x / x!\) \(\lambda\) \(\lambda\) Rare event counts
Gaussian\((\mu,\sigma^2)\) \(\frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}\) \(\mu\) \(\sigma^2\) CLT limit, noise
Exponential\((\lambda)\) \(\lambda e^{-\lambda x}\), \(x\ge0\) \(1/\lambda\) \(1/\lambda^2\) Waiting times
Beta\((\alpha,\beta)\) \(\propto x^{\alpha-1}(1-x)^{\beta-1}\) \(\frac{\alpha}{\alpha+\beta}\) \(\frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}\) Prior over probabilities

Relationship The Poisson distribution arises as the limit of Binomial\((n,p)\) when \(n\to\infty\), \(p\to 0\) with \(\lambda = np\) fixed. This justifies its use for rare events in large populations.

0.7 Statistical Estimation

From observations to parameters: bias, consistency, and the likelihood framework.

0.8 Maximum Likelihood Estimation

Likelihood Function Given i.i.d. observations \(\mathbf{x} = (x_1,\ldots,x_n)\) from a parametric family \(\{f(\cdot;\theta): \theta\in\Theta\}\), the likelihood and log-likelihood are: \[L(\theta) = \prod_{i=1}^{n} f(x_i;\theta), \qquad \ell(\theta) = \sum_{i=1}^{n} \log f(x_i;\theta).\] The MLE is \(\hat\theta_{\mathrm{MLE}} = \arg\max_{\theta\in\Theta} \ell(\theta)\).

Example — Gaussian MLE For \(X_i \sim \mathcal{N}(\mu,\sigma^2)\), the log-likelihood is \[\ell(\mu,\sigma^2) = -\tfrac{n}{2}\log(2\pi\sigma^2) - \tfrac{1}{2\sigma^2}\sum_{i=1}^n(x_i-\mu)^2.\] Solving \(\partial\ell/\partial\mu = 0\) and \(\partial\ell/\partial\sigma^2 = 0\): \[\hat\mu = \bar{x}, \qquad \hat\sigma^2 = \tfrac{1}{n}\sum_{i=1}^n(x_i - \bar{x})^2.\] Note: \(\hat\sigma^2\) is biased; the unbiased version divides by \(n-1\).

Asymptotic Properties of MLE Under regularity conditions, as \(n\to\infty\):

  1. Consistency: \(\hat\theta_n \xrightarrow{p} \theta^*\)
  2. Asymptotic normality: \[\sqrt{n}(\hat\theta_n - \theta^*) \xrightarrow{d} \mathcal{N}(0,\, \mathcal{I}(\theta^*)^{-1})\] where \(\mathcal{I}(\theta)\) is the Fisher information.
  3. Efficiency: The MLE achieves the Cramér–Rao lower bound asymptotically.

0.9 Bias–Variance Decomposition

Theorem — Bias–Variance Decomposition For an estimator \(\hat\theta\) of \(\theta^*\), the Mean Squared Error decomposes as: \[\underbrace{\mathbb{E}[(\hat\theta - \theta^*)^2]}_{\text{MSE}} = \underbrace{(\mathbb{E}[\hat\theta] - \theta^*)^2}_{\text{Bias}^2} + \underbrace{\mathbb{E}[(\hat\theta - \mathbb{E}[\hat\theta])^2]}_{\text{Variance}}.\]

0.9.1 In supervised learning

For a model \(\hat{f}\) predicting \(Y = f(X) + \varepsilon\) with \(\varepsilon \sim (0,\sigma^2)\):

\[\mathbb{E}[(Y - \hat{f}(x))^2] = \underbrace{\operatorname{Bias}^2[\hat{f}(x)]}_{\text{underfitting}} + \underbrace{\operatorname{Var}[\hat{f}(x)]}_{\text{overfitting}} + \sigma^2.\]

The \(\sigma^2\) term is the irreducible error — no model can eliminate it.

Practical Implications - High bias: model too simple; misses structure. Use richer hypothesis class or more features. - High variance: model too complex; memorises noise. Use regularisation, early stopping, or more data. - Regularisation (Ridge/Lasso) introduces deliberate bias to reduce variance — a controlled tradeoff.

0.11 Limit Theorems

The Law of Large Numbers and Central Limit Theorem — the twin pillars underpinning statistical inference and Monte Carlo methods.

0.12 LLN & Central Limit Theorem

Strong Law of Large Numbers If \(X_1, X_2, \ldots\) are i.i.d. with \(\mathbb{E}[|X|] < \infty\), then: \[\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \;\xrightarrow{\text{a.s.}}\; \mu = \mathbb{E}[X].\]

Almost sure convergence: \(\mathbb{P}(\lim_{n\to\infty}\bar{X}_n = \mu) = 1\).

Monte Carlo Integration The LLN justifies estimating \(I = \mathbb{E}[g(X)] = \int g(x)f(x)dx\) by: \[\hat I_n = \frac{1}{n}\sum_{i=1}^n g(X_i), \quad X_i \overset{\text{i.i.d.}}{\sim} f.\] Error \(\hat{I}_n - I = O(n^{-1/2})\) regardless of dimension — unlike quadrature.

Central Limit Theorem If \(X_1,\ldots,X_n\) are i.i.d. with mean \(\mu\) and variance \(\sigma^2 < \infty\): \[\frac{\bar X_n - \mu}{\sigma/\sqrt{n}} \;\xrightarrow{d}\; \mathcal{N}(0,1) \quad \text{as } n\to\infty.\] Equivalently, \(\bar X_n \;\dot\sim\; \mathcal{N}(\mu, \sigma^2/n)\) for large \(n\).

Confidence Interval from CLT A \((1-\alpha)\) CI for \(\mu\) (known \(\sigma\)) is: \[\bar X_n \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}},\] where \(z_{\alpha/2} = \Phi^{-1}(1 - \alpha/2)\). For \(\alpha=0.05\): \(z_{0.025} \approx 1.96\).

0.13 Computing with Distributions — Python

0.13.1 scipy.stats & numpy

import numpy as np
from scipy import stats

# Gaussian: pdf, cdf, quantile
rv = stats.norm(loc=0, scale=1)
rv.pdf(0)      # ≈ 0.3989
rv.cdf(1.96)   # ≈ 0.9750
rv.ppf(0.975)  # ≈ 1.9600  (z-score)

# Sample mean & CLT illustration
rng   = np.random.default_rng(42)
X     = rng.exponential(scale=2, size=(10_000, 50))
means = X.mean(axis=1)          # shape (10000,)

# By CLT: means ~ N(2, 4/50)
print(means.mean(), means.std())
# → ~2.000  ~0.283  (theory: √(4/50)≈0.283)

0.13.2 MLE with scipy

from scipy.stats import norm

# Fit a Gaussian to data (MLE)
data = rng.normal(5, 2, 500)
mu_hat, sig_hat = norm.fit(data)
print(f"μ̂={mu_hat:.3f}, σ̂={sig_hat:.3f}")
# → μ̂≈5.00, σ̂≈2.00

# Log-likelihood at the MLE
ll = norm.logpdf(data, mu_hat, sig_hat).sum()

# Bootstrap 95% CI for the mean
boots = [rng.choice(data, 500).mean()
         for _ in range(5000)]
ci = np.percentile(boots, [2.5, 97.5])
print(f"95% CI: [{ci[0]:.3f}, {ci[1]:.3f}]")

0.14 Hypothesis Testing Framework

Setup - Null hypothesis \(H_0\): default assumption (e.g. \(\mu = \mu_0\)) - Alternative \(H_1\): what we seek evidence for - Test statistic \(T(\mathbf{X})\): summarises data - Rejection region \(\mathcal{C}\): \(H_0\) rejected iff \(T \in \mathcal{C}\)

p-value \[p = \mathbb{P}_{H_0}(T \ge t_{\text{obs}})\]

The probability, under \(H_0\), of observing data at least as extreme as what was observed. Reject \(H_0\) iff \(p \le \alpha\).

\(H_0\) true \(H_0\) false
Reject \(H_0\) Type I error (\(\alpha\)) Correct (Power)
Retain \(H_0\) Correct (\(1-\alpha\)) Type II error (\(\beta\))

One-sample z-test (known \(\sigma\)) \[T = \frac{\bar X - \mu_0}{\sigma/\sqrt{n}} \;\overset{H_0}{\sim}\; \mathcal{N}(0,1).\]

Reject \(H_0\) at level \(\alpha\) (two-sided) if \(|T| > z_{\alpha/2}\). Power against \(\mu_1\): \[1-\beta = \Phi\!\left(-z_{\alpha/2} - \frac{|\mu_1-\mu_0|}{\sigma/\sqrt{n}}\right).\]

Common Misinterpretations \(p < 0.05\) does not mean \(H_0\) is false, nor that the effect is large or practically important. Report effect sizes and CIs alongside \(p\)-values.

0.15 Lecture Summary & Reading

0.15.1 Key takeaways

  • A random variable maps probability space to \(\mathbb{R}\); characterised by PMF/PDF and CDF.
  • Expectation is linear; variance follows its own algebra — understand these cold.
  • Know the standard families (Gaussian, Poisson, Exponential, Beta) and their DS applications.
  • MLE maximises log-likelihood; for regular families it is consistent and asymptotically efficient.
  • The Bias–Variance decomposition governs every learning algorithm’s error.
  • The CLT justifies Gaussian approximations and Monte Carlo estimation; LLN ensures convergence.
  • Hypothesis testing: prespecify \(\alpha\), compute \(p\)-value, do not confuse statistical with practical significance.
